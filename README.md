# 🚀 AI Job Preparation Platform

An AI-powered full-stack web application that helps job seekers prepare for interviews by analyzing resumes, generating interview questions, creating personalized preparation plans, and creating ATS-friendly resumes using Google Gemini AI.

## ✨ Features

- 📄 Resume Analysis
- 🤖 AI-Powered Interview Question Generation
- 📊 Personalized Interview Preparation Plan
- 📑 ATS-Friendly Resume Generator
- 🔐 User Authentication
- ☁️ MongoDB Atlas Database
- ⚡ Fast and Responsive UI

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Sass
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Google Gemini API
- JWT Authentication
- Puppeteer
- Zod

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/raj6436/Ai-Job-Preparation-Platform.git
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

### Backend

```bash
cd Backend
npm install
npm run dev
```

## 🔑 Environment Variables

Create a `.env` file inside the Backend folder.

```
GOOGLE_GENAI_API_KEY=YOUR_API_KEY
MONGO_URI=YOUR_MONGODB_CONNECTION_STRING
```
